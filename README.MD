#### Project Setup  

Create a new project using static web template.

```sh
$ file
$   new project
$   static web static web 
```

Create src and test directories.

```sh
$ file
$   new directory
$   src 

$ file
$   new directory
$   test 
```



#### Package Installation

```sh
$ npm init
$ npm i mocha
$ npm i chai
$ npm i axios
$ npm install
```


#### Run your tests

```sh
$ npm run test
```
